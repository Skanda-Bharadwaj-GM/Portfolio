# Portfolio – Skanda Bharadwaj GM

A Next.js portfolio with About, Intro (contacts), Projects, Resume, and a GitHub button.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

1. **Profile links & phone**  
   Edit `src/lib/profile.ts` and set your LinkedIn, Instagram, GitHub URLs and phone number.

2. **About me**  
   Edit `src/app/page.tsx` and replace the placeholder paragraph with your text.

3. **Intro text**  
   Edit the short intro in `src/app/intro/page.tsx`.

4. **Projects**  
   Edit `src/app/projects/page.tsx`: replace the placeholder array with your projects (title, description, href, tags).

5. **Resume**  
   Put your resume PDF in `public/resume.pdf`. The Resume page will show it and offer a download link.

## Build

```bash
npm run build
npm start
```
